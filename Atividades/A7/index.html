O que é SEO

SEO é a sigla em inglês para Search Engine Optimization, que significa otimização para mecanismos de busca. Utilizando a SEO, busca-se alavancar a visibilidade de um site, para que os mecanismos de busca, como Google e Bing, o mostre como um dos primeiros resultados, aumentando assim o fluxo de visitantes do site.


Conteúdo

Antes de mais nada, é importante dizer que um dos fatores que mais influenciam a relevância de um site, a despeito das técnicas propostas pelo SEO, é o conteúdo. Isso se dá pois conteúdos relevantes tendem a ser indicados por outras pessoas e também linkados por outros sites, o que aumenta as chances que uma determinada página seja indicada por um mecanismo de busca.


Título

O título é um dos pontos mais importantes do SEO. É importante que o título do site contenha as palavras chaves do conteúdo do site, para facilitar a busca pelos buscadores e que o resultado apareça para as pessoas que o procuram. O título também deve ser sucinto, pois há um limite de caracteres que os buscadores apresentam nos resultados, após isso o título é “cortado”. Além disso, cada página do site, não apenas a principal, deve ter um título único que a identifique, seguindo os mesmos critérios citados anteriormente.


Cabeçalhos

Cabeçalhos ajudam à pessoa que utiliza o site a navegar por ele e encontrar seu conteúdo. Deve ser sucinto e apontar sobre o que se trata. Idealmente, deve haver apenas um cabeçalho < h1 >. Os demais podem ser usados mais de uma vez, conforme a importância e prioridade que tenham.


Meta tags

As meta tags são as descrições do site. Nela devem estar contidas a descrição do site, utilizando as mesmas palavras chave do título, pois reforçam aos buscadores sobre o que ele se trata e adiciona consistência ao seu conteúdo.


Canonical tags

As tags canônicas são marcações inseridas no código de páginas da web para definir que elas são um conteúdo original. É importante para não deixar páginas duplicadas, pois prejudica os mecanismos de busca.


Imagens

As imagens devem ser descritas em um texto alt. Os textos alt, além de serem importantes para o funcionamento de leitores utilizados por deficientes visuais, ajudam os buscadores a identificar do que se trata uma determinada imagem e alavancar a visibilidade do site.


Links

Os links internos e externos precisam ser adequadamente indexados, com palavras chave que se refiram a eles, pois isso também ajuda os buscadores.
